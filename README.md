Up Consultancy - Web Geliştirme Projem
Archis Academy eğitimim sırasında geliştirdiğim, tasarımından kodlamasına kadar her aşamasında tek başıma çalıştığım bir danışmanlık sitesi projesi.

Neler Yaptım?
Bu projeyi listedeki diğerlerinden ayıran şey, ekip arkadaşı beklemeden tüm modülleri kendi başıma tamamlamış olmam:

Figma: Figma dosyasındaki tasarımı, piksellere ve renk paletine sadık kalarak sıfırdan HTML ve CSS ile giydirdim.

Kod: Hiçbir hazır kütüphane veya React gibi frameworkler kullanmadım. Tamamen temel HTML5, CSS3 ve JavaScript ile mantığını kurdum.

Mobil Uyumluluk: Sitenin sadece bilgisayarda değil, telefonda da düzgün görünmesi için responsive tasarım üzerine bayağı kafa yordum.
